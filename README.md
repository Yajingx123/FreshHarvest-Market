# FreshHarvest-Market

## Project Overview
PHP Web Application Deployment on Linux Server

FreshHarvest Market is a database-driven web application designed to support
the operations of a fresh produce market, including inventory management,
order processing, and supplier coordination.

This system was developed as part of the DI31003 Database Systems coursework.

## Tech Stack
- Linux (Cloud Server)
- Nginx
- PHP-FPM
- MySQL
- Shell Script

## Deployment

1. Install Nginx
2. Install PHP-FPM
3. Configure server block
4. Deploy project files
5. Reload services
6. Change config/db_connect.php to your mysql password
